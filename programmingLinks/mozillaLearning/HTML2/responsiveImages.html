Responsive Images

We can use different image files and declare at what width they should be used to fix the issue of images showing funky on the page. The image below uses the <img> element and that element's srcset and sizes attributes. Although, this doesn't seem to want to use the "kero192w.png" image provided, only the full resolution "kero384w.png" image.

Alternatively, the below image uses the <picture> element.

This element also directs the browser to download and display the correct image before showing the page, possibly reducing bandwidth.
