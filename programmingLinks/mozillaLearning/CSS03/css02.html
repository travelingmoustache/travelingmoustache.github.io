Styling Backgrounds in CSS

Background Colors

background-color
background-color determines the color of the background of any element. This has many available property values and a long shorthand.
background-image
Prints an image accross the background of an element. The image is not automatically sized to its container and will pattern by default. If both background-color and background-image are used, the image will appear ontop the background color.
background-repeat
Determines how the image repeats. no-repeat, repeat-x, repeat-y, and repeat (default) are available as property values.
background-size
A property available for ensuring the background image fits the element it is contained in. It takes the property values cover, which covers the container with the image and may trim it, and contain, which covers the container with the image the best it can without trimming the image and while maintaining its aspect ratio. This property may also use length and percentage values.
background-position
Uses both, or a mix of, keywords (top, right) and length and percentage values to position the image on the background with respect to the top left corner of the container. The values are given in the x then y order. For example, the following code would position the top left corner of the image 20% accross and 50% down its container from the top left: background-position: 20% 50%;. background-position can also take four values at a time, where the first and third values are top, right, bottom, or left, and the second and fourth values are how far the background image is to be positioned away from those side of the container.
Gradient Backgrounds
Backgrounds that show a color gradient. This uses the background-image property. There is a little too much to get into with notes here, but there is a page for making background gradients here, and Mozilla's examples can be viewed here.
Using More Than One Background
background-image can accept more than one input with commas separating inputs. Each other background property (background-repeat, background-size, etcetera) can take multiple values as well. The values given to background properties affect the background in the order they were given. Meaning, the fourth background-image will be affected by the fourth background-size value given.
background-attachment
Determines how the background scrolls when the page is scrolled. Three values are available for this property:
scroll
Scrolls with the page, but does not scroll if the element it is inside of is scrolled.
fixed
Does not scroll with the page or when the element is scrolled. It is always static.
local
Causes the background to be fixed to the element it is contained in; the background will scroll when the element is scrolled, and will scroll with the page as long as the element is on the page still.
A helpful page exemplifying these three values can be found here.
Background Shorthand
To list background values using shorthand, each background must be listed with each of their properties in order that they would appear. See this Mozilla page for more information on background shorthand.
Borders
Borders have both long and shorthand properties. border can set the width, style, and color for the entire border. border-(width/style/color) can set any of those border property values individually, and border-(top/right/bottom/left) can set an individual border side. These can be combined into the longest way of declaration, border-(top/right/bottom/left)-(width/style/color).
border-radius
border-radius changes the way the border wraps around an element. Usually this is used for making rounded corners. border-(top-right, bottom-left, etcetera)-radius can be used for setting the values of specific corners. Either one or two property values can be given: the first value defines the horizontal radius and the second the verticle radius; one value defines both radiuses.
