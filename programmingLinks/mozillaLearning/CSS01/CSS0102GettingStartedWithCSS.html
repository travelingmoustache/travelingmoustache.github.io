Selectors reference:

Consider that there are <h1>, <p>, and <body> elements, as well as a special class in our document. If we use the following selector in our CSS:
<body> <h1> + <p> .special {...}
This would select each element with the special class that is inside a <p> element that is immediately preceded by an <h1> element that is inside a <body> element.

CSS Definitions

Selector
Shown here as li.special. This select all <li> elements with the special class.

CSS Declaration

CSS Declaration Blocks

CSS Ruleset
